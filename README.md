# Emmanuel de Mazieres — Portfolio

A refined, minimal industrial design portfolio inspired by high-end studios such as LAYER (layerdesign.com/projects/).

**Logo / Name:** Emmanuel de Mazieres

## Design Direction

- Clean, spacious, typography-led layout
- Soft warm off-white background (#f7f6f3)
- High-contrast near-black text
- Project presentation inspired by LAYER’s projects page: clear project name + client/category
- Elegant typography (Instrument Sans + Inter)
- Subtle hover effects and scroll fade-ins
- Fully responsive

## Sections

- **Hero** — Confident, short introduction
- **Selected Work** — Vertical list of projects with image placeholders
- **About** — Bio and key details
- **Contact** — Direct email + links

## How to Customize

1. **Projects**  
   Replace the gradient placeholders with real project photography.  
   Update the project names and categories.  
   Point the links to individual case studies if you have them.

2. **About**  
   Edit the three paragraphs and the detail rows (location, focus, previous experience).

3. **Contact**  
   Change the email and LinkedIn / portfolio links.

4. **Colors**  
   Adjust the CSS variables at the top of `css/styles.css` if you prefer a cooler or darker palette.
